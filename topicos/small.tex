\section{Texto e Comandos} % Este comando faz o título da seção.

Palavras são separadas por um ou mais espaços. Paragrafos
são separados por uma ou mais linhas em branco. A saída não
é afetada por espaços extras ou por linhas em branco extras.

\subsection{Um aviso}  % Este comando faz o título da subseção.

Lembre-se, não escreva os 10 caracteres especiais
%                &   $   #   %   _   {   }   ^   ~   \
\& \$ \# \% \_ \{ \} \textasciicircum \ \textasciitilde
\ \textbackslash \ exceto como um comando! Eles são impressos
com os comandos
%Este comando inicia um ambiente ``verbatim''(ao pé da letra)
\begin{verbatim}
\& \$ \# \% \_ \{ \} \textasciicircum \textasciitilde \textbackslash.
\end{verbatim}   %Este comando termina o ambiente ``verbatim''.
                 %Tudo que estiver dentro do ambiente é impresso
                 %exatamente como é digitado
A maioria dos comandos do \LaTeX \ são iniciados com o
caracter \textbackslash. Uma \textbackslash \ sozinha produz
um espaço.
