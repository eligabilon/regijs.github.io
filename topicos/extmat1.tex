\section*{Ementa}
Introdução ao MATLAB, Introdução ao programa de editoração
eletrônica de textos matemáticos \LaTeX, Matemática
Financeira e Aplicações de Álgebra Linear com o uso do
MATLAB.

\section*{Programa}
\begin{enumerate}
\item Introdução ao MATLAB.
Vamos mostrar como este programa pode fazer o computador
trabalhar como uma máquina de calcular científica
sofisticada.
\item Introdução ao \LaTeX.
Este é o programa usado pelos matemáticos, físicos e outros cientistas
para escrever os artigos científicos. É o programa adequado
para editoração eletrônica de textos que contenham fórmulas.
Ele pode ser usado também para escrever home pages.
\item Matemática Financeira.
Vamos mostrar entre outras coisas como encontrar os valores
das prestações em financiamentos que estão sendo práticados
hoje no mercado, como a tabela price, o sistema SAC e
sistema SACRE, usado pelo sistema financeiro da habitação.
Além disso, vamos mostrar como calcular a taxa de juros que
está realmente sendo praticada nos financiamentos.
\item Aplicações de Álgebra Linear.
%\begin{enumerate}
Vamos estudar como podemos  construir curvas e superfícies
que passam por pontos dados. Ou como no caso em que os
pontos são obtidos experimentalmente, como usar Quadrados
Mínimos e Splines para encontrar curvas e superfícies suaves
que melhor representam os pontos.

%\end{enumerate}
\end{enumerate}
