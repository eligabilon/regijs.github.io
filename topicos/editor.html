Instruções:

  • Você pode escrever a sua página usando a sintaxe do LATEX.

  • Não é necessário escrever o preâmbulo.

  • Apertando o botão view page, você vê como está a página gerada pelo que você está digitando. Se quiser gravar o resultado basta escolher no menu do browser File -> Save as. Escolhendo no menu Go -> Back você pode voltar à esta página.

  • Apertando o botão view source, uma janela é aberta e você vai ver uma cópia do que foi digitado. Escolhendo no menu do browser File -> Save as você pode gravar a fonte em LATEX da sua página para edição no futuro.

  • Para ler um arquivo que foi escrito em LATEX e convertê-lo em HTML, basta abrí-lo em um editor de textos (p. ex. Note-Pad), marcar o seu conteúdo (p. ex. CONTROL+A) copiar no clipboard (p. ex. com CONTROL+C) e colar na caixa de texto deste editor p. ex. com CONTROL+V).

  • Para a inclusão de figuras, pode ser usado o comando \includegraphics{arquivo.gif}, para figuras no formato gif.

  • Para referências a outras páginas, pode ser usado o comando \href{end URL}{nome da referência}. Por exemplo, página principal do departamento foi gerado com
    \href{http://www.mat.ufmg.br}{página principal do departamento}.
