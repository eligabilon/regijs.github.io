O \LaTeX\ é um programa de domínio púplico, por isso existem
várias implementações. Aqui está uma implementação de 32 bits
chamada
Mik$\!$\TeX, uma interface gráfica com um editor de
textos chamada, \TeX Shell, e um visualizador, \textbf{YAP}. 
Todos
estes programas são ``freeware'', isto é, são gratuítos e
podem trabalhar juntos como se fossem um só, mas, para que
isto aconteça, é necessário configurá-los. Estamos
disponibilizando uma versão já configurada. 
Além destes programas aqui estão também:
\begin{enumerate}
\item
\href{gsview}{\textbf{GhostView}} 
para
visualizar e imprimir arquivos no formato Postscript.
\item
\href{ar40eng.exe}{\textbf{Acrobat Reader}} para visualizar e imprimir 
arquivos no formato pdf.
\item
\href{ltxmc108.exe}{\textbf{\LaTeX Macros}} que fornece uma interface
gráfica para comandos do \LaTeX.
\end{enumerate}

\begin{center}
{\Large Instalação}
\end{center}

\begin{itemize}
\item Instale o Ghostview. Para isto 
faça o download dos arquivos que estão na pasta
\href{gsview}{gsview} e grave-os todos numa mesma pasta. 
Depois, basta clicar duas vezes com
o botão esquerdo do mouse no arquivo \texttt{setup.exe}.
\item Descompacte os arquivos \href{foiltex.zip}{foiltex.zip},
\href{contrib.zip}{contrib.zip} e
\href{atalhos.zip}{atalhos.zip} a partir da raiz
\texttt{c:$\backslash$} com a opção ``Use Folder Names''
(no caso do descompactador ser o WINZIP, clique duas vezes em cada um
dos arquivos, clique no botão \textbf{Extract}, depois em
\textbf{Extract to:} escreva \texttt{c:$\backslash$} e marque
\textbf{Use Folder Names}).
\item Faça o ``download'' dos
arquivos que estão na pasta \texttt{miktex} e grave-os
todos numa mesma pasta. Para isto siga as instruções que
estão no arquivo \href{miktex/index.html}{index.html}.
Instale o Mik\!\TeX. Para isto basta clicar duas vezes com
o botão esquerdo do mouse no arquivo \texttt{setupwiz.exe}.
\begin{enumerate}
\item
Quando perguntado por
"the local TEXMF tree", escolha `No local TEXMF tree'.
\item
Quando perguntado por uma lista de ``additional TEXMF root directories",
marque `No preexisting TEXMF directory trees'. 
\end{enumerate}
\item
Acrescente a linha

\texttt{set PATH=\%PATH\%;c:$\backslash$texmf$\backslash$miktex$\backslash$bin}

no arquivo \texttt{autoexec.bat}
do seu computador (ele fica em \texttt{c:$\backslash$}). Para isso, clique
no arquivo \texttt{autoexec.bat} com o botão direito do mouse e escolha
\textbf{Editar}, desta forma o arquivo será aberto no Bloco 
de Notas.

\item
Se desejar, instale o programa
\href{ltxmc108.exe}{\textbf{\LaTeX Macros}} que fornece uma interface
gráfica para comandos do \LaTeX.

\end{itemize}



Para começar a usar o \LaTeX\ você deve clicar em \textbf{Iniciar}, depois
em \textbf{Programas}, depois em \textbf{MikTeX} e finalmente clicar em
\textbf{TexShell}.
