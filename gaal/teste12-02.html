Considere o sistema linear que depende do parâmetro $a\in\mathbb{R}$. \[ \left\{ \begin{array}{rrrrrrr} x &+& y &+& z &=&2\\ x &+& y &+& 2z &=& 4\\ && && (a^2-9)z &=& a-3.\\ \end{array} \right. \] Podemos afirmar que
Se $a\neq \pm 3$, então o sistema tem uma única solução.
Se $a\neq 3$ e $a\neq -5/2$, então o sistema não tem solução.
Somente se $a= 3$ o sistema tem infinitas soluções.
Se $a=-5/2$, então o sistema não tem solução.
