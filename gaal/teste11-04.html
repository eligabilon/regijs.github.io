Considere o sistema linear que depende do parâmetro $a\in\mathbb{R}$. \[ \left\{ \begin{array}{rrrrrrr} x &+& 2y &+& z &=&3\\ x &+& y &-& z &=& 2\\ x &+& y &+& (a^2-5)z &=& a.\\ \end{array} \right. \] Podemos afirmar que
Se $a\neq \pm 2$, então o sistema tem uma única solução.
Se $a=-2$, então o sistema tem solução.
Se $a=\pm 2$, então o sistema tem infinitas soluções.
Se $a=2$, então o sistema não tem solução.
